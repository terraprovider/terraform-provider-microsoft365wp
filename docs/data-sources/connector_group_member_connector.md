# microsoft365wp_connector_group_member_connector (Data Source)

Use this resource to add/assign an existing connector to a connector group.

Notes:  
  - When deleting this resource, the connector will be assigned back to the default connector group.  
  - To import this resource, an ID consisting of `connector_group_id` and `id` being joined by a forward slash (`/`) must be used.
